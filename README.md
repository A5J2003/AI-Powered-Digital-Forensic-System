# AI-Powered Digital Forensic System

> A multimodal deepfake detection and forensic analysis framework for verifying the authenticity of digital media using AI, explainability, and forensic principles.

---

##  Overview

This system performs **comprehensive forensic analysis** on multimedia inputs by combining:

- Video analysis  
- Audio analysis  
- Text analysis  
- Cross-modal consistency  

It not only detects deepfakes but also provides **explainable evidence** and **forensic-grade reporting**.

---

##  Key Features

###  Video Deepfake Detection
- XceptionNet (CNN-based spatial analysis)
- Swin Transformer (attention-based detection)
- Frame-level anomaly scoring
- Grad-CAM visualization

### Audio Deepfake Detection
- CRNN (CNN + RNN hybrid)
- Log-Mel spectrogram processing
- Temporal pattern analysis
- Audio Grad-CAM

### Text AI Detection
- RoBERTa & DeBERTa models
- Detection of AI-generated linguistic patterns
- Confidence-based classification

### Cross-Modal Forensics
- ASR (Audio Speech Recognition)
- OCR (Text extraction from frames)
- ASR-OCR consistency scoring
- Detects dubbing & tampering

---

## Explainability Layer

-  Grad-CAM heatmaps (video & audio)
-  Attention visualization (Swin Transformer)
-  Modality-wise probability charts
-  Human-readable forensic explanations

---

##  Digital Forensics Capabilities

- SHA-256 hashing (file integrity verification)
- Chain-of-custody logging
- Metadata extraction and tamper detection
- Evidence traceability
